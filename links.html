To access the code for the ABM model as shown in this website, follow the link here to my Github respository. The scripts are python and are best ran in Anaconda Spyder. To run the model select run file in the top toolbar to intialise the code. A GUI window should pop-up. On this GUI window, select run model in the top left-hand corner. Happy Coding!
