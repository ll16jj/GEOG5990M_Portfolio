Property Search Map


Below is an interactive map of Leeds, West Yorkshire, where each marker is the location of a property currently for sale, where general information is provided such as, location, price, type of property, number of bedrooms, number of bathrooms, number of receptions(living/dining rooms), and a link to that property on Zoopla for further information/pictures. Use the price function underneath the interactive map to filter for different price ranges, and click the reset button to reset the map. The data displayed below were all sourced and appended between 29th-30th April 2022.
