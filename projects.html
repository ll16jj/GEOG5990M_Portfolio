My Projects


I have produced various projects during my time studying GIS at the Univerisity of Leeds. Underneath, click on the images for a brief description and to link to that relevant project. There are two projects, the first is an Agent-Based Model project produced as a part of Assessment 1 for, GEOG 5990M Programming for Geographical Information Analysis: Core Skills. This project involves producing a computational model where agents/objects move and interact with each other and their environments. The second project contributes to the final project work of GEOG 5870M Web-Based GIS, which involves an website of residential properties for sale throughout Leeds. This project has general information for those wanting to invest or buy a property in Leeds such as, general demographic data, burglary rates, average property price per MSOA and Electoral Wards, property prices per LSOA over the past 20 years, as well as an interactive map, where each property has been placed on a googlemap as a marker with relevant information (price, address, number of bedrooms/bathrooms/receptions, link to that property on Zoopla and type of property).
